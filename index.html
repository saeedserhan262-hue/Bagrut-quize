سؤال 12

01

إيجاد أساس المتوالية

في متوالية هندسية: \[ a_1 = 3,\quad a_4 = 24 \] أوجدي أساس المتوالية \(r\).

\[ a_4 = a_1 r^3 \Rightarrow 24 = 3r^3 \Rightarrow r^3 = 8 \Rightarrow r = 2 \]
02

Trick بسيط

في متوالية هندسية: \[ a_2 = 10,\quad a_5 = 80 \] أوجدي \(r\).

\[ a_5 = a_2 r^3 \Rightarrow 80 = 10r^3 \Rightarrow r^3 = 8 \Rightarrow r = 2 \]
03

مجموع حدّين متتاليين

إذا كان: \[ a_1 + a_2 = 12,\qquad a_2 + a_3 = 24 \] أوجدي \(r\).

نفرض \(a_1 = x\) إذًا: \[ a_2 = xr,\quad a_3 = xr^2 \] ومنه: \[ x(1+r)=12 \] \[ xr(1+r)=24 \] بقسمة المعادلة الثانية على الأولى: \[ r = 2 \]
04

حدّ سالب مخادع

في متوالية هندسية: \[ a_1 = 5,\quad r = -2 \] أوجدي \(a_4\).

\[ a_4 = a_1 r^3 = 5(-2)^3 = 5(-8) = -40 \] الــ trick هنا في الإشارة السالبة.
05

إيجاد الحد الأول

في متوالية هندسية: \[ r = 3,\quad a_5 = 486 \] أوجدي \(a_1\).

\[ a_5 = a_1 r^4 \] \[ 486 = a_1 \cdot 3^4 = 81a_1 \] \[ a_1 = \frac{486}{81} = 6 \]
06

مجموع متوالية نهائية

أوجدي مجموع المتوالية اللانهائية: \[ a_1 = 7,\quad r = \frac{1}{3} \]

بما أن: \[ |r| < 1 \] إذًا المجموع موجود: \[ S_\infty = \frac{a_1}{1-r} = \frac{7}{1-\frac13} = \frac{7}{\frac23} = \frac{21}{2} \]
07

هل يوجد مجموع لا نهائي؟

هل يوجد مجموع لا نهائي للمتوالية: \[ a_1 = 2,\quad r = -3 \] ؟

لا. لأن شرط وجود مجموع لا نهائي هو: \[ |r| < 1 \] وهنا: \[ |-3| = 3 > 1 \] إذًا لا يوجد مجموع لا نهائي.
08

مقارنة سريعة

في متوالية هندسية موجبة الحدود: \[ a_3 = 18,\quad a_4 = 54 \] أوجدي \(a_1\).

أولًا: \[ r = \frac{a_4}{a_3} = \frac{54}{18} = 3 \] ثم: \[ a_3 = a_1 r^2 \Rightarrow 18 = 9a_1 \Rightarrow a_1 = 2 \]
09

سؤال فيه انتباه للترتيب

إذا كان: \[ a_1 = 4,\quad r = \frac12 \] أوجدي: \[ a_1 + a_2 + a_3 \]

\[ a_2 = 2,\quad a_3 = 1 \] إذًا: \[ a_1 + a_2 + a_3 = 4+2+1=7 \]
10

إيجاد عدد الحدود

في متوالية هندسية: \[ a_1 = 2,\quad r = 2 \] إذا كان: \[ a_n = 64 \] أوجدي \(n\).

\[ a_n = a_1 r^{n-1} \] \[ 64 = 2 \cdot 2^{n-1} = 2^n \] \[ 64 = 2^6 \Rightarrow n=6 \]
11

مجموع أول 4 حدود

في متوالية هندسية: \[ a_1 = 1,\quad r = 3 \] أوجدي: \[ S_4 \]

\[ S_4 = \frac{a_1(r^4-1)}{r-1} = \frac{1(3^4-1)}{3-1} = \frac{81-1}{2} = \frac{80}{2} = 40 \]
12

سؤال ذكي جدًا

في متوالية هندسية: \[ a_1 + a_3 = 20,\qquad a_2 = 6 \] أوجدي \(r\) إذا كان \(r>0\).

نفرض: \[ a_1 = x,\quad a_2 = xr,\quad a_3 = xr^2 \] ومعطى: \[ xr = 6 \Rightarrow x = \frac{6}{r} \] وأيضًا: \[ x + xr^2 = 20 \] نعوض: \[ \frac{6}{r} + \frac{6}{r}r^2 = 20 \] \[ \frac{6}{r} + 6r = 20 \] نضرب بـ \(r\): \[ 6 + 6r^2 = 20r \] \[ 3r^2 - 10r + 3 = 0 \] \[ (3r-1)(r-3)=0 \] إذًا: \[ r=\frac13 \quad \text{أو} \quad r=3 \] وكلاهما موجب.
